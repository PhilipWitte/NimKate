Nimrod programming language support for Kate text editor.

Install
---
- **Nimrod**: Copy `nimrod.xml` to `~/.kde4/share/apps/katepart/syntax`
- **Colors**: Kate > Settings > Configure Kate > Fonts & Colors > (Schema) Import...
